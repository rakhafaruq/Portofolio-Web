Tentang Saya

Selamat datang di portofolio saya! Saya adalah seorang mahasiswa yang bersemangat dalam mempelajari teknologi yang inovatif dan fungsional. Dengan pengalaman di bidang pengembangan web dan pengolahan data, saya suka mengubah ide-ide kompleks menjadi aplikasi yang sederhana dan menarik bagi pengguna.

Keahlian utama saya meliputi HTML, CSS, JavaScript, dan berbagai kerangka kerja modern seperti React. Saya selalu antusias untuk belajar teknologi baru dan menerapkannya dalam proyek-proyek saya.

Proyek Pilihan Saya

Aplikasi Database Berbasis Website

Membangun sebuah aplikasi web fungsional yang terintegrasi dengan database MySQL. Aplikasi ini dirancang untuk memudahkan pengelolaan dan monitoring data melalui fitur CRUD (Create, Read, Update, Delete) yang dikembangkan menggunakan Python.

Teknologi: Python, MySQL, HTML

Lihat di GitHub

Aplikasi Prediksi Penyakit Jantung

Sebagai Ketua Kelompok, saya memimpin pengembangan sistem untuk memprediksi penyakit jantung menggunakan Machine Learning. Proyek ini menerapkan algoritma K-Means Clustering untuk segmentasi pasien dan Regresi Logistik untuk prediksi, dengan hasil yang disajikan melalui dashboard interaktif menggunakan Streamlit.

Teknologi: Python, Machine Learning, Streamlit
